# image-scale

Scale an image using nearest neighbour interpolation.


## Installation

    make eggs
    make


## Usage

    ./image-scale IN_PATH OUT_PATH OUT_WIDTH OUT_HEIGHT


## Dependencies

- CHICKEN 5
